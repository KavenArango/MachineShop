    Machines

    Bellow are all the machines we have to offer

    Bridgeport

    The Bridgeport milling machine is used to machine a stock material into an object through user input. This 3-axes machine can use many different milling ends and is equipped with a hands-free system to make it easier to mill in a straight line. Milling is the process of machining an object—commonly stock material—using rotary cutters to remove excess material.

    Lathe

    The lathe uses rotational motion to mill a stock material—that is, by turning or boring the material—into an object through user input. Milling is the process of machining an object—commonly stock material—using rotary cutters to remove excess material.

    HAAS VF2

    The HAAS CNC milling machine is used to mill an object using computer code, specifically G- or M-code. This is the larger of the two CNC machines at CETA. Milling is the process of machining an object—commonly stock material—using rotary cutters to remove excess material.

    Syil

    The Syil CNC milling machine is used to mill an object using computer code, specifically G- or M-code. This is the smaller of the two CNC machines at CETA. Milling is the process of machining an object—commonly stock material—using rotary cutters to remove excess material.
